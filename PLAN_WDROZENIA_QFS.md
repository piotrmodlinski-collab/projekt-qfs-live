# QFS - Plan Realizacji i Wdrozenia

## 1. Pozycjonowanie (Wykonane)
- Strona ustawiona pod studio gamedev z dwoma liniami biznesowymi:
  - Produkcje wlasne (IP)
  - Outsourcing / Co-Development

## 2. Architektura informacji (Wykonane)
- Start
- Produkcje
- Outsourcing
- Zespol
- Kontakt

## 3. Materialy video i portfolio (Wykonane)
- Podpiete gameplay MP4 na podstronie Produkcje
- Rozdzielone prezentacje: wlasne IP vs delivery partnerow

## 4. Standard profesjonalny (Wykonane)
- Sekcje wiarygodnosci: standard pracy, pipeline, modele wspolpracy
- Jasne CTA: brief, rozmowa projektowa, rekrutacja
- Lepsza semantyka i nawigacja aktywnej podstrony

## 5. SEO i gotowosc publikacyjna (Wykonane)
- Meta opisy + OpenGraph na wszystkich podstronach
- robots.txt
- sitemap.xml

## 6. Checklista przed publikacja (Do wykonania po Twojej stronie)
- Podmiana finalnych emaili kontaktowych (jesli inne niz obecne)
- Weryfikacja finalnych URL pod docelowy routing serwera
- Podpiecie analityki (GA4 / Matomo)
- Finalny test mobile + Lighthouse + Core Web Vitals

## 7. Kolejny krok
- Deploy na hosting + podpiecie domeny
- Po wdrozeniu: szybki audit produkcyjny 24h po publikacji
